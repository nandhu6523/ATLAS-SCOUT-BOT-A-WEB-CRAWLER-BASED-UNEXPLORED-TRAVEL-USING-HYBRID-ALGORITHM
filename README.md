## ATLAS SCOUT BOT : A WEB CRAWLER BASED UNEXPLORED TRAVEL USING HYBRID ALGORITHM

ATLAS Scout Bot is a web crawler–based travel recommendation system that uses a hybrid algorithm to suggest unexplored and offbeat destinations based on user preferences.

## About

ATLAS Scout Bot is a web crawler–based travel recommendation system that helps users discover unexplored and less-crowded travel destinations. It collects travel data from websites, blogs, and reviews using automated web crawling techniques. A hybrid algorithm analyzes user preferences and location popularity to generate personalized recommendations. The project aims to promote offbeat tourism and enhance travel planning through intelligent data analysis.

## Features

- Automated web crawling to collect travel data from blogs, websites, and reviews
- Hybrid recommendation algorithm for personalized and offbeat destination suggestions
- User preference–based filtering (budget, location, interests, season, etc.)
- Focus on unexplored and less-crowded travel destinations
- Simple and user-friendly interface for easy travel discovery

## Requirements

- Programming Language: Python
- Web Crawling Libraries: BeautifulSoup, Scrapy, Requests
- Backend Framework: Flask
- Database: MySQL / SQLite
- Frontend: HTML, CSS, JavaScript
- Internet connection for data crawling and updates
